Home

Welcome to the Web Technology course website. This site is designed to guide you through the fundamental and advanced concepts of building web applications using a variety of tools and languages including HTML, CSS, XML, Perl, PHP, and MySQL. Each module is structured to enhance both theoretical understanding and hands-on skills in web development.

Syllabus

Module I – INTRODUCTION (9 Hours)

XHTML, HTML Forms, Frames, Tables, Lists, Images, Multimedia, CSS – Inline, Internal, External – Font, Color, Padding, Margin, Border, Background, Class, ID, Element Positioning.

Module II – XML (9 Hours)

SGML, XML, Syntax of XML, XML document structure, XML namespaces, XML Schemas and DTD, DOM and SAX parsers, Displaying XML documents in browsers, XSLT.

Module III – PERL (9 Hours)

PERL Introduction, Data Types and Variables, Scalars, Arrays, Hashes, Operators, Control Structures, Regular Expressions, File Input and Output, Writing CGI Scripts, Forms and CGI, Cookies and Sessions in PERL.

Module IV – PHP (9 Hours)

Introduction to PHP, Syntax, Variables, Data Types, Operators, Control Structures, Functions, Arrays, Form Handling, File Handling, Cookies, Sessions, Error Handling.

Module V – MySQL (9 Hours)

Introduction to MySQL, Creating Databases and Tables, SQL Queries – SELECT, INSERT, UPDATE, DELETE, PHP and MySQL Integration, Connecting to MySQL Server, Performing CRUD Operations.

Lab Manual

  • Create a personal website using HTML and CSS with proper structure and styling.
  • Create XML documents with DTD and Schema validation.
  • Write a Perl script to display system date and time on a webpage.
  • Develop a simple PHP form to accept user input and display it.
  • Build a login and registration system using PHP and MySQL.
  • Perform CRUD operations using PHP connected to a MySQL database.
